# MES 2.1.3 Release Notes

MES 2.1.3은 외부 오버레이나 다른 페이지 UI가 나중에 추가되어도 MES UI가 최상단을 유지하도록 보강한 패치입니다.

## 변경 사항

- MES 버튼, 패널, 선택 캡처 레이어, 토스트를 문서 끝의 전면 UI 묶음으로 재정렬
- 같은 최대 z-index를 쓰는 외부 고정/오버레이 UI가 추가될 때 MES UI를 다시 전면으로 복구
- UI 전면 유지 주기를 단축해 가려지는 시간을 줄임
- 외부 오버레이가 MES 버튼을 덮는 상황을 검증하는 smoke test 추가

## 검증

- 원본/미니파이 문법 검사
- UI 잡음 검사
- UI 전면 유지 smoke test
- 전체 smoke test
