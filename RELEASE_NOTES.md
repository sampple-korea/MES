# MES Update Notes

## 2.1.3

- 외부 오버레이나 다른 페이지 UI가 나중에 추가되어도 MES UI가 전면을 회복하도록 보강했습니다.
- MES 버튼, 패널, 선택 캡처 레이어, 토스트를 문서 끝의 전면 UI 묶음으로 재정렬했습니다.
- 같은 최대 z-index를 쓰는 외부 고정/오버레이 UI가 추가될 때 MES UI를 다시 전면으로 복구합니다.
- UI 전면 유지 주기를 단축해 가려지는 시간을 줄였습니다.
- 외부 오버레이가 MES 버튼을 덮는 상황을 검증하는 smoke test를 추가했습니다.

## 2.1.2

- iframe 내부에서는 MES 버튼, 패널, 토스트, UI 스타일을 생성하지 않도록 변경했습니다.
- iframe 내부에서도 저장된 차단 규칙과 DOM 변경 감시는 계속 적용합니다.
- 같은 출처 iframe 내부 요소는 부모 페이지 UI에서 더 안정적으로 탐색하도록 보강했습니다.
- iframe 작업자 모드 회귀 테스트를 추가했습니다.

## 2.1.1

- 설정에 UI 언어 선택을 추가했습니다.
- 한국어, 영어, 중국어, 일본어 UI를 지원합니다.
- 기본 언어는 한국어로 유지합니다.
- 선택 패널, 설정, 규칙 목록, 인스펙터, 토스트, 진단/리소스 라벨의 주요 문구를 언어팩 기반으로 정리했습니다.
- 선택 모드에서 광고 링크, iframe, 고정 레이어가 실제 클릭으로 넘어가는 누수를 줄이기 위해 투명 캡처 레이어와 이벤트 차단을 강화했습니다.
- 한국어 기본 README와 영어/중국어/일본어 문서를 정리했습니다.
- 언어 기본값, 언어 저장, 다국어 초기 로딩, 선택 모드 클릭 누수 방지 검증을 추가했습니다.

## 2.1.0

- 동적 광고 ID, class, 리소스 URL, 링크 패턴을 분석하는 고급 차단 후보를 추가했습니다.
- 고급 후보도 기존 MES 후보 목록 안에서 매칭 수, 위험도, 미리보기, 저장 확인 흐름을 그대로 거치도록 했습니다.
- 설정에 저사양 모드를 추가했습니다.
- `MES.min.js` minified 배포 파일을 추가했습니다.
- 설치/업데이트 URL을 GitHub raw의 `MES.min.js`로 변경했습니다.
- `npm test`가 minified 파일을 빌드하고 source/minified 양쪽 smoke test를 모두 실행하도록 검증 흐름을 강화했습니다.

## 2.0.1

- userscript 매니저가 GM API를 전역 객체 속성이 아니라 sandbox 바인딩으로 제공하는 경우도 감지하도록 수정했습니다.
- 레거시 MES 저장소 식별자를 유지해 업데이트 후 기존 GM 저장소 규칙이 사라진 것처럼 보이는 문제를 줄였습니다.
- GM 저장소가 정상으로 잡힌 뒤에도 예전 사이트별 `localStorage` fallback 규칙이 남아 있으면 자동으로 GM 저장소에 병합합니다.
- GitHub raw 주소를 설치/업데이트 URL로 사용하도록 메타데이터와 README를 정리했습니다.
- README 기본 문서를 한국어로 바꾸고, 기능별 실제 UI 캡처 갤러리를 추가했습니다.
- 기존 필터 데이터가 감지될 때 마이그레이션하는 방법을 README에 이미지와 함께 추가했습니다.
- sandbox GM 저장소 감지와 fallback 규칙 마이그레이션 경로를 smoke test에 추가했습니다.

## 2.0.0

- UI를 더 작고 정돈된 컨트롤로 바꿨습니다.
- 낮은 대비의 원형 런처, 스위치형 설정, 자석식 패널 정렬, 태블릿 레이아웃 대응을 추가했습니다.
- 선택 모드는 기본 전체 패널로 열리며, 필요할 때만 컴팩트 모드로 전환됩니다.
- 컴팩트 모드에서도 상위/하위 요소 조절 슬라이더와 핵심 액션을 유지합니다.
- 런처 방식은 버튼 모드와 제스처 모드 중 하나를 고르는 구조로 단순화했습니다.
- 제스처 모드에서는 손가락 수와 탭 횟수를 바로 설정할 수 있습니다.
- 요소 상위/하위 탐색, 선택자 후보, 저장 규칙 관리, 동적 감시, 차단 무결성 복구, UI 자가 복구, 인스펙터, 마이그레이션, 백업/복원을 전반적으로 강화했습니다.
- 안내 메시지와 주요 UI에서 장식성 이모지를 제거했습니다.
- 메인 대시보드의 과한 브랜드 노출을 줄이고, 제품 표기는 설정 하단의 작은 정보로 이동했습니다.
