# MES 2.1.1 Release Notes

MES 2.1.1은 UI 언어 지원을 확장한 마이너 업데이트입니다.

## 변경 사항

- 설정에 UI 언어 선택을 추가했습니다.
- 한국어, 영어, 중국어, 일본어 UI를 지원합니다.
- 기본 언어는 한국어로 유지합니다.
- 선택 패널, 설정, 규칙 목록, 인스펙터, 토스트, 진단/리소스 라벨의 주요 문구를 언어팩 기반으로 정리했습니다.
- 선택 모드에서 광고 링크, iframe, 고정 레이어가 실제 클릭으로 넘어가는 누수를 줄이기 위해 투명 캡처 레이어와 이벤트 차단을 강화했습니다.
- README는 한국어를 기본으로 유지하고, 영어/중국어/일본어 문서를 별도로 추가했습니다.
- `npm test`에 언어 기본값, 언어 저장, 영어/중국어/일본어 초기 로딩, 선택 모드 클릭 누수 방지 검증을 추가했습니다.

## 검증

- `npm test` 통과 필요
- GitHub Actions Validate 통과 대상: `main`, `stable`
