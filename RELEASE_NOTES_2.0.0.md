# MES 2.0.0 Release Notes

MES 2.0.0은 1.x 계열을 모바일 요소 선택기답게 다시 다듬은 메이저 업데이트입니다. 목표는 작은 화면에서도 덜 가리고, 규칙은 더 안전하게 만들고, 동적 페이지에서도 차단이 오래 유지되게 하는 것입니다.

## 기존 MES보다 좋아진 점

- UI가 더 작고 정돈된 Apple 스타일 컨트롤로 바뀌었습니다. 낮은 대비의 원형 런처, 스위치형 설정, 자석식 패널 정렬, 태블릿 레이아웃 대응을 추가했습니다.
- 선택 모드는 기본 전체 패널로 열리며, 필요할 때만 컴팩트 모드로 전환됩니다. 컴팩트 모드에서도 상위/하위 요소 조절 슬라이더와 핵심 액션을 유지합니다.
- 런처 방식은 버튼 모드와 제스처 모드 중 하나를 고르는 구조로 단순화했습니다. 제스처 모드에서는 손가락 수와 탭 횟수를 바로 설정할 수 있습니다.
- 요소 상위/하위 탐색이 개선되었습니다. 부모, 자식, 하위 후보를 더 안정적으로 탐색하고 현재 선택 범위를 더 쉽게 조절할 수 있습니다.
- 선택자 생성이 더 다양해졌습니다. 정밀 선택자, 유사 패턴, 속성 기반, 힌트 기반, 클래스 기반, 리소스 기반 후보를 제공합니다.
- 저장 전 선택자 품질과 위험도를 표시하고 확인합니다. 너무 넓은 선택자, 위치 의존 선택자, 복잡한 선택자, 안정적인 선택자, 정밀 선택자를 구분합니다.
- 저장 규칙 관리가 강화되었습니다. 검색, 규칙별 활성/비활성, 현재 사이트 정리, 현재 매치 수, 사이트 범위 표시, 영향 미리보기, 저장 직후 실행 취소를 지원합니다.
- 동적 감시 성능과 안정성이 좋아졌습니다. DOM 변경, open Shadow DOM, class/id/hidden 속성 변경 뒤에도 저장 규칙을 다시 적용합니다.
- 차단 무결성 복구가 추가되었습니다. 페이지가 숨김 스타일, 규칙 style 노드, adopted stylesheet, CSSOM 규칙을 건드려도 MES가 다시 복구합니다.
- UI 자가 복구가 강화되었습니다. 페이지가 대시보드 노드, ID, class, hidden 상태, inline style, UI stylesheet를 제거하거나 바꿔도 다시 살아납니다.
- 인스펙터 기능이 넓어졌습니다. 요소 요약, 선택자 후보, HTML, 계산된 CSS, 스크립트 힌트, 페이지 소스, 쿠키, 리소스, 진단, 포커스 모드, 개인정보 보호 복사를 제공합니다.
- 기존 저장 데이터가 감지될 때만 마이그레이션 메뉴가 보이며, 가져온 규칙은 중복 제거 후 병합됩니다.
- 백업/복원이 저장 규칙, 비활성 규칙, 설정을 함께 보존합니다.
- 안내 메시지와 주요 UI에서 장식성 이모지를 제거해 더 깔끔하게 보이도록 했습니다.
- 메인 대시보드의 과한 브랜드 노출을 줄이고, 제품 표기는 설정 하단의 작은 정보로 이동했습니다.

## 검증

- `npm test`로 문법 검사, UI 노이즈 검사, Playwright 기반 모바일/태블릿 smoke flow를 검증합니다.
- smoke test는 모바일 폭, 태블릿 portrait/landscape, 제스처 설정, 컴팩트 계층 조절, 자석식 정렬, 저장 규칙 미리보기, 실행 취소, 오래된 규칙 정리, 선택자 후보, 선택자 위험도, 동적 차단 복구, Shadow DOM 재적용, UI 자가 복구를 포함합니다.
